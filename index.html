Know Chords

Aplicativo Flutter para estudo harmonico, montagem de progressoes e transposicao automatica por tom, com foco em uso rapido para violao/guitarra.

Secoes

1. Visao Geral 2. Funcionalidades 3. Arquitetura 4. Regras de Negocio 5. Stack Tecnologica 6. Como Executar 7. Qualidade e Testes 8. Estagio

1. Visao Geral

O sistema permite selecionar um tom, visualizar acordes diatonicos e montar uma sequencia de forma interativa.

A transposicao ocorre automaticamente quando o tom base muda, mantendo o padrao harmonico por indices de grau.

2. Funcionalidades Implementadas

  • Selecao de tom entre 12 opcoes.
  • Renderizacao dos acordes disponiveis para o tom selecionado.
  • Montagem de sequencia com repeticao de acordes.
  • Remocao individual de acordes da sequencia.
  • Acao para limpar a sequencia completa.
  • Tela "Salvos" preparada para futura persistencia.

3. Arquitetura Atual

Entrada e Navegacao

main.dart inicializa o app e controla as abas Home e Salvos via NavigationBar.

Dados Harmonicos

data/acordes.dart define tons e o mapeamento de acordes por tom.

Telas

screens/home_screen.dart concentra o fluxo principal; screens/salvos_screen.dart e placeholder.

Widgets

widgets/ contem componentes reutilizaveis: seletor de tom e lista de acordes.

4. Regras de Negocio

  1. A sequencia e armazenada por indice e nao por nome fixo de acorde.
  2. Ao trocar o tom, o sistema aplica os mesmos indices no novo conjunto de acordes.
  3. Sem tom selecionado, a lista de acordes fica oculta.
  4. Duplicatas na sequencia sao permitidas por design.

5. Stack Tecnologica

  • Flutter + Dart
  • Material 3 para UI
  • Dependencias principais: cupertino_icons
  • Dependencias de dev: flutter_test, flutter_launcher_icons

6. Como Executar

Com Flutter configurado no ambiente, execute:

Builds comuns:
